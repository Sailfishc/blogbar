目前有非常多的博客写作方式，Blogbar 将它们分为 2 大类：博客平台、独立博客。

博客平台

根据定位的不同，博客平台可分为大众型博客平台、专业型博客平台、轻博客。

大众型博客平台包括 新浪博客、网易博客、天涯博客 等，它们都涌现于 Web 2.0 时代，有很多资深用户常年在其中写作。随着互联网的发展，新一代博客平台陆续发布，比如 简书、国外的 Medium，它们都有着非常不错的用户体验，干净清新，专注于内容本身。 值得一提的还有新兴博客平台 FarBox，它可以将本地的文本文件转化为网页，将本地的照片转化为网络相册，让写博客变得轻松和自然。

而专业型博客平台，则一般是针对某一特定行业的写作空间，比如针对程序员交流技术的 CSDN 和 博客园，针对学术界的 科学网，都有着浓厚的行业气息。

最后则是轻博客，它是介于博客与微博之间的一种博客服务。代表产品有 点点网、网易的 LOFTER、新浪的 Qing，以及国外的轻博客产品 Tumblr。

独立博客

独立博客，意味着需要博主自行提供博客服务器。根据是否动态生成博客内容，可将其分为：动态博客、静态博客。

最著名的动态博客程序当属 WordPress，它是目前世界上使用最广泛的博客系统，有着庞大的用户、活跃的社区以及丰富的模板/插件资源。 国内也有非常不错的博客系统 Typecho，经过三年沉淀，愈加轻量高效。 而 Ghost 则是另一款内容写作发布工具，纯粹干净、简洁沉静。不过如果想要自行部署 Ghost，需要具备一定的技术经验。

静态博客一般会在本地生成博客所需的全部页面代码，然后上传到网络服务器上即可。相对于动态博客来说，静态博客更加轻量，无需配置数据库，服务器响应速度会更快。静态博客系统的代表有 Jekyll、Octopress 和 Hexo。 如果你懂 Git，可无需另外购买服务器，直接将生成的静态文件推送到 GitHub Pages 上，GitHub 会免费为你托管博客内容。 若你觉得还不够 Geek，而且酒香不怕巷子深，也可以直接在 GitHub 上通过开 Issue 的方式写博客。

再好的博客平台、再易用的博客程序，也抵不过一篇原创博文。

Keep writing，勿忘初心。
